# Cadastro de Livros

Este é um projeto simples feito para praticar **HTML, CSS e JavaScript**.

A aplicação permite cadastrar livros informando **ID, título e ano de publicação**.
Os livros cadastrados aparecem em uma tabela e também podem ser **excluídos** ou **buscados pelo título**.

## Funcionalidades

* Cadastrar livros
* Listar livros em uma tabela
* Buscar livro pelo título
* Apagar livro da lista

## Tecnologias utilizadas

* HTML
* CSS
* JavaScript

## Estrutura do projeto

```
cadastro_livro
│
├── index.html
├── style.css
├── README.md
├── index.js
```

## Como usar

1. Baixe ou clone o repositório.
2. Abra o arquivo **index.html** no navegador.
3. Preencha os campos e clique em **Cadastrar Livro**.

## Objetivo

Este projeto foi desenvolvido para praticar **manipulação do DOM com JavaScript**, criação de elementos HTML e uso de arrays para armazenar os dados dos livros.
